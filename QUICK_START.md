## 📋 Common Workflows

### Workflow A: Fill My Wall Perfectly
**Goal**: Image exactly fills my 2m × 1.5m wall

1. **Measure wall**: 2m wide, 1.5m tall
2. **Set wall dimensions**: 200cm × 150cm
3. **Upload image**: Any resolution works!
4. **App does**: Resizes image → Calculates pages → Shows grid
5. **Result**: 6 pages (3×2) that fit perfectly
6. **Export**: PDF → Print → Hang

### Workflow B: Use All of High-Res Image
**Goal**: Print 5000×4000px photo at best quality

1. **Don't set wall dimensions** (skip this)
2. **Upload image**: 5000×4000px
3. **Choose page size**: A3 (bigger pages = fewer to print)
4. **Result**: Image split into 4×3 = 12 A3 pages
5. **Joined size**: 2100×1680mm
6. **Export**: PDF → Print

### Workflow C: Multiple Photos on Wall
**Goal**: Arrange 4 photos on 2m × 2m wall

1. **Combine photos**: Stitch into one image 
2. **Set wall**: 200cm × 200cm
3. **Upload**: Combined image
4. **Get**: Optimal page grid (maybe 4×4 = 16 pages)
5. **Export**: Print all pages

## ✅ What Gets Calculated Automatically

When you set wall dimensions and upload an image:

| Step | What Happens |
|------|-------------|
| 1 | Reads your wall dimensions |
| 2 | Gets image aspect ratio |
| 3 | Calculates optimal size to fill wall |
| 4 | Resizes image (maintains aspect) |
| 5 | Determines page grid needed |
| 6 | Calculates total joined size |
| 7 | Checks if it fits (✓/⚠️/❌) |
| 8 | Shows everything in UI |

---

## 🛠️ What Each Setting Does

### Wall Dimensions
- **Width**: Horizontal wall space
- **Height**: Vertical wall space  
- **Unit**: cm, m (meters), or ft

**Effect**: Triggers automatic image scaling

### Page Size (A4, A3, Letter, Legal)
- **A4**: Standard, smallest (most pages)
- **A3**: Double size (fewer pages)
- **Letter**: US standard
- **Legal**: US tall

**Effect**: Fewer pages with larger page size

## 🎨 Tips for Best Results

### Before You Start:
- ✓ Measure wall accurately in cm or m
- ✓ Use high-resolution image (3000px minimum)
- ✓ Choose appropriate page size (A3 for fewer pages)

### During Setup:
- ✓ Set wall dimensions FIRST (before uploading)
- ✓ Upload a single image (no splitting before)
- ✓ Wait for automatic scaling to complete

### After Export:
- ✓ Print on same paper type (all glossy or all matte)
- ✓ Use sharp utility knife for clean cuts
- ✓ Align overlap marks carefully
- ✓ Use spray adhesive for secure mounting

## ❓ FAQ

**Q: Can I change wall size later?**  
A: Yes! Update dimensions, app recalculates instantly.

**Q: What if wall is non-rectangular?**  
A: Use the largest rectangular area available.

**Q: Can I use portrait (tall) mode?**  
A: Yes, wall orientation doesn't matter, app adapts.

**Q: Can I manually set page count?**  
A: Currently auto-only. Override by not setting wall dimensions.

**Q: What about odd wall sizes?**  
A: Any size works: 147cm × 89cm, no problem!

**Q: Does scaling reduce quality?**  
A: Only if scaling UP dramatically. For down-scaling, quality improves (less data needed).

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| Image seems small | Check if scaled down. Set larger wall size |
| Too many pages | Use A3 instead of A4, or larger wall |  
| Image looks distorted | Aspect ratio preserved - check wall proportions |
| Pages don't fit | Export Panel shows how much to adjust |
| Scaling looks odd | Verify wall measurements in correct unit |
